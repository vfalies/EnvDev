## Requirements

Two system dependencies are required :

- Docker
- Docker Compose

## Installation

```shell
git clone git@github.com:vfalies/EnvDev.git
```

Create a `.env` file to define your configuration.
An example is available with `.env.dist` file.

To install environment

```shell
make install
```
